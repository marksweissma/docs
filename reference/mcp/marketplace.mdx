---
title: marketplace
description: Manage plugin files in connected GitHub repositories
---

Read, update, and commit plugin files stored in a GitHub-backed marketplace repository. This tool is used by plugin developers who are building or maintaining Moxn plugins.

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `action` | enum | Yes | The operation to perform |
| `marketplaceId` | string | Conditional | Marketplace ID — required for most actions |
| `branchName` | string | No | Git branch name (default: `main`) |
| `pluginSlug` | string | No | Plugin slug for filtering `list_plugins` |
| `componentPath` | string | Conditional | Full file path within the repo — required for `read_component`, `update_component` |
| `content` | string | Conditional | File content — required for `update_component` |
| `message` | string | Conditional | Commit message — required for `commit` |
| `fromBranch` | string | No | Source branch for `create_branch` |

## Actions

| Action | Notes |
|--------|-------|
| `list_marketplaces` | List all marketplaces in the workspace |
| `list_plugins` | List plugins in a marketplace; filter by `pluginSlug` |
| `read_component` | Read a file's content by path |
| `update_component` | Update a file's content (creates working state in the repo) |
| `commit` | Commit all uncommitted changes to Git |
| `list_branches` | List Git branches in the marketplace repo |
| `create_branch` | Create a new Git branch |

## Workflow

```
1. list_marketplaces({})                     → get marketplaceId
2. list_plugins({marketplaceId})             → discover plugins and file paths
3. read_component({marketplaceId, componentPath, branchName})  → read a file
4. update_component({marketplaceId, componentPath, branchName, content}) → edit it
5. commit({marketplaceId, branchName, message: "Update skill description"}) → push to GitHub
```

## Examples

```
"List the available marketplaces"
→ marketplace({action: "list_marketplaces"})

"Show me all plugins in marketplace abc"
→ marketplace({action: "list_plugins", marketplaceId: "abc"})

"Read the SKILL.md for the code-review plugin"
→ marketplace({action: "read_component", marketplaceId: "abc", componentPath: "plugins/code-review/skills/review/SKILL.md", branchName: "main"})

"Update the plugin description and commit"
→ marketplace({action: "update_component", marketplaceId: "abc", componentPath: "plugins/code-review/MANIFEST.md", branchName: "my-update", content: "..."})
→ marketplace({action: "commit", marketplaceId: "abc", branchName: "my-update", message: "Update code-review plugin description"})
```
