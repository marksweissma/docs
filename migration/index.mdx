## Two Ways to Run

All import and export commands are available through two tools:

| Tool | Install | Command prefix |
|------|---------|----------------|
| **`@moxn/kb-cli`** | `npm install -g @moxn/kb-cli` | `knowledge-base import-notion`, `knowledge-base export-local`, etc. |
| **`@moxn/kb-migrate`** | No install needed | `npx @moxn/kb-migrate notion`, `npx @moxn/kb-migrate export`, etc. |

Both use the same underlying migration engine. The CLI is convenient if you already have `@moxn/kb-cli` installed for other commands. `npx @moxn/kb-migrate` works without any prior installation.

## Date Filtering

All commands support date filters to narrow the scope of your import or export:

```bash
# Only import/export docs modified in the last week
--modified-after=2026-02-27

# Only docs created in January
--created-after=2026-01-01 --created-before=2026-02-01
```

Dates use ISO 8601 format (`YYYY-MM-DD` or full datetime).
